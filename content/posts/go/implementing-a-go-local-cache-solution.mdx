Cache can be used for multiple scenarios, specially when you want to have a layer between you service and a database or a second service that return always the same information for period of time, like and API or something like that. In some cases can be necessary to use a distributed cache like Redis where you will have a cluster to store your data, it happens specially when you are working with multiple services running at the same time, many replicas like pods or tasks. Considering that you have the necessity to control the cache locally in the application before go to an external service. Or if you have an unavailability service that you cant reach out, a local cache can be a good fit to that situation, allowing your service keep working without an external information for a time, until the principal service being available. So thinking about that i decided to share an Go example of implementation of local cache. Lets breaking it down.

## The principal structure
First of all, we will create a simple struct that will be used as our template to store data in our memory. Its simples, we just have an value witch type is an `interface{}` type of data.


``` go
type localCacheItem struct {
    value interface{}
    expiresAt time.Time
}
```

The second important part is, our cache needs to be safe when we have multiples executions concurrently consuming it. Here, we will use Mutex to ensure that we have a mutual exclusion lock. The lock can be held by an arbitrary number of readers or a single writer, it ensure that all threads are getting the same information. You can read more about Mutex here.

``` go
type LocalCache struct {
    mu sync.RWMutex
    items map[string]*localCacheItem
    ttl time.Duration
}
```

As you can se, to complement the explanation about the struct we have an map where the key type is string and or value type is our localCacheItem, and to finish we have an ttl (Time To Live) expiration, to define after how much time our registry will expire.

## Methods

We will have four methods: Get, Set, Clean and CleanExpired. There are some implementations that is commom between them. In each method we start from locking our transaction before get, set or clean our data, to prevent  inconsistence between our asynchronous services, and when the process finishes using defer we ensure that the process was unlock.

## The Get method

Here we try to retrieve our data by the key and if it doesn't exists or the found item is expired we just return a negative (false) information to the caller, otherwise we return the information that have been found in the cache.

``` go
func (c *LocalCache) Get(key string) (interface{}, bool) {
   c.mu.RLock()
   defer c.mu.RUnlock()
  
   item, exists := c.items[key]
   if !exists {
    return nil, false
   }
  
   if time.Now().After(item.expiresAt) {
    return nil, false
   }
  
   return item.value, true
}
```

## The Set Method

That is really simple, we just create a key in our map setting our key value with a expiration time defined.

``` go
func (c *LocalCache) Set(key string, value interface{}) {
   c.mu.Lock()
   defer c.mu.Unlock()
  
   c.items[key] = &localCacheItem{
    value:     value,
    expiresAt: time.Now().Add(c.ttl),
   }
}
```

## The Clean Method

Here we have the option to clean our cache and reset all information inside the in memory map. It's like a restart.

``` go
func (c *LocalCache) Clear() {
   c.mu.Lock()
   defer c.mu.Unlock()
  
   c.items = make(map[string]*localCacheItem)
}
```

## The CleanExpired Method

That is a very important method, we can't just leave all data inside the memory without control. If you have a service that process millions of data, you won't leave it in the memory of your service, because of that we defined an expiration date, to remove all expired data from the service memory.

``` go
func (c *LocalCache) CleanExpired() {
   c.mu.Lock()
   defer c.mu.Unlock()
  
   now := time.Now()
   for key, item := range c.items {
    if now.After(item.expiresAt) {
     delete(c.items, key)
    }
   }
}
```

## The implementation

Now you can see all the implementation here.

``` go
package cache

import (
 "sync"
 "time"
)

type localCacheItem struct {
    value interface{}
    expiresAt time.Time
}

type LocalCache struct {
    mu sync.RWMutex
    items map[string]*localCacheItem
    ttl time.Duration
}

func New(ttl time.Duration) *LocalCache {
   return &LocalCache{
    items: make(map[string]*localCacheItem),
    ttl:   ttl,
   }
}

func (c *LocalCache) Get(key string) (interface{}, bool) {
   c.mu.RLock()
   defer c.mu.RUnlock()
  
   item, exists := c.items[key]
   if !exists {
    return nil, false
   }
  
   if time.Now().After(item.expiresAt) {
    return nil, false
   }
  
   return item.value, true
}

func (c *LocalCache) Set(key string, value interface{}) {
   c.mu.Lock()
   defer c.mu.Unlock()
  
   c.items[key] = &localCacheItem{
    value:     value,
    expiresAt: time.Now().Add(c.ttl),
   }
}

func (c *LocalCache) Clear() {
   c.mu.Lock()
   defer c.mu.Unlock()
  
   c.items = make(map[string]*localCacheItem)
}

func (c *LocalCache) CleanExpired() {
   c.mu.Lock()
   defer c.mu.Unlock()
  
   now := time.Now()
   for key, item := range c.items {
    if now.After(item.expiresAt) {
     delete(c.items, key)
    }
   }
}
```
